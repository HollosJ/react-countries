# React Countries Database

This application allows the user to view various pieces of information about any country on the [RESTCountries API](https://restcountries.eu/).

## Data

#### Current

- Name
- Flag
- Language
- Capital
- Language
- Region/Subregion

#### Future data

- Currency
- ISO Code(s)
- Calling code
- Latitude/Longitude
- Regional Bloc(s)
